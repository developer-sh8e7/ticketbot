# بوت الإدارة والألعاب المتقدم

بوت ديسكورد شامل مبني باستخدام TypeScript ومكتبة discord.js v14، يجمع بين أنظمة إدارة السيرفرات والألعاب التفاعلية.

## المميزات
- **نظام الإدارة (System):** رسائل الترحيب، الوداع، الرتب التلقائية، نظام التحقق (Verification)، قوانين السيرفر، السجلات (اللوق)، وإشعارات الإدارة.
- **الألعاب (Games):** أسئلة عامة (Trivia)، سرعة كتابة، الرقم المحظوظ، روليت السحب، سلسلة الكلمات، تخمين الكلمة، صراحة وتحدي، ولعبة القنبلة الموقوتة.

## المتطلبات
- Node.js إصدار 18 أو أحدث
- توكن بوت ديسكورد (Bot Token) و Client ID
- بيئة TypeScript

## طريقة التركيب والتشغيل

1. تثبيت الحزم والمكتبات المطلوبة:
   \`\`\`bash
   npm install
   \`\`\`

2. ضبط المتغيرات البيئية (إنشاء ملف \`.env\` أو إضافتها في استضافة مثل Railway):
   \`\`\`env
   TOKEN=توكن_البوت
   CLIENT_ID=ايدي_البوت
   GUILD_ID=ايدي_السيرفر
   \`\`\`

3. بناء الكود (Compile):
   \`\`\`bash
   npm run build
   \`\`\`

4. رفع الأوامر لسيرفر الديسكورد (Slash Commands):
   \`\`\`bash
   npm run deploy
   \`\`\`

5. تشغيل البوت:
   \`\`\`bash
   npm start
   \`\`\`

## قائمة الأوامر المتاحة
### الإدارة
- \`/setup\` - لوحة إعداد قنوات النظام (الترحيب، اللوق، التحقق، الرتب التلقائية) بضغطة زر.
- \`/rules\` - لعرض القوانين، إضافتها، أو حذفها.

### الألعاب
- \`/trivia\` - أسئلة ثقافية مع احتساب الفائز.
- \`/typerace\` - اختبار سرعة الكتابة (WPM).
- \`/lucky\` - خمن الرقم المحظوظ.
- \`/roulette\` - عجلة اختيار عشوائي بين الأعضاء.
- \`/wordchain\` - سلسلة الكلمات المترابطة.
- \`/guess\` - لعبة تخمين الحروف للكلمة المجهولة.
- \`/truthordare\` - اختار صراحة أو تحدي.
- \`/bomb\` - قنبلة موقوتة، مررها قبل ما تنفجر!
