# Content and Copy Report

## Voice and Tone

عربي سعودي طبيعي، واضح ومهني بدون لغة تقنية زائدة او وعود مبالغ فيها.

## Messaging Principles

- ابدأ باسم الخدمة والسعر مباشرة.
- استخدم جمل قصيرة داخل التفاعل.
- وضح ان السعر يشمل التصميم والبرمجة والتسليم حسب نطاق الباقة.
- CTA مباشر: `ابدأ مشروعك`.
- حافظ على الاسعار والمزايا الحالية بدون اختراع خدمات جديدة.

## Page-level Copy

| Page/Area | Copy Goal | Example Copy |
|---|---|---|
| Card | تعريف سريع | `متجر إلكتروني` + `٣٬٥٠٠ ر.س` |
| Interaction hint | تعليم بدون شرح طويل | `اضغط على البطاقة لاطلاق التفاصيل` |
| Detail panel | قرار سريع | وصف مختصر + اول ثلاث مزايا |
| Primary CTA | بدء التحويل | `ابدأ مشروعك` |
| Secondary CTA | حالة خارج الباقات | `أو اطلب عرض مخصص` |
| Discount | قيمة مباشرة | `وفر 22%` |

## CTA System

| CTA Type | Example | Usage |
|---|---|---|
| Primary | ابدأ مشروعك | داخل لوحة كل باقة |
| Secondary | اطلب عرض مخصص | اسفل المسار |
| Filter | متاجر إلكترونية | تضييق العرض فقط |
| Close | إغلاق تفاصيل الباقة | label لقارئ الشاشة |

## Empty / Loading / Error Copy

- Loading مرئي بدون نص حتى لا يضيف ضوضاء.
- WebGL error ينتقل تلقائيا الى بطاقات HTML ولا يحتاج رسالة مخيفة.
- اذا صارت القائمة فارغة مستقبلا: `الباقات تحت التحديث، تقدر تطلب عرض مخصص الحين.`

## Onboarding / Help Copy

- Desktop: `اضغط على البطاقة لاطلاق التفاصيل`.
- Mobile: نفس النص واضح والضغط يعمل مباشرة.
- السطر `المسار يتحرك بشكل مستمر` يفسر الحركة بدون مصطلحات تقنية.

## Phrases to Avoid

- `احك لنا عن فكرتك`.
- `حلول ثورية لا مثيل لها`.
- `انطلق في رحلة التحول الرقمي`.
- تفاصيل تقنية مثل WebGL وshader في واجهة العميل.
- وعود مدة ثابتة لا تتوافق مع عبارة التقدير الحالية.
